# LS MES

## Build Setup

```bash
# install dependencies
$ npm install

# 로컬
$ npm run dev

# 운영기
$ npm run devp

# 빌드
$ npm run build

$ npm publish

$ npm run start

# generate static project
$ npm run generate
```

For detailed explanation on how things work, check out [Nuxt.js docs](https://nuxtjs.org).
